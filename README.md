# OpenAi Assistants API Template For Telegram

1. Clone the repo
2. create new venv
3. initiate venv
4. install requirements with -> `pip install -r requirements.txt`
5. create .env file
   OPENAI_API_KEY=sk-xxxxxxx
   TELEGRAM_BOT_TOKEN=xxxxxx
